In this section, we will make sure you can open the Git Bash prompt that you just installed. To launch it:

  • Click the Windows or Start icon
  • In the Programs list, open the Git folder
  • Click the option for Git Bash.

(Don't be distracted by the "Git GUI" program. In this tutorial, we use "Git Bash.")
